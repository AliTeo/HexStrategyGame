The Hexagonal Strategy Game now has a fully functional SFML 3.0.2 graphical
client alongside the original text-based client!

WHAT'S NEW:
================================================================================

✅ VISUAL HEX GRID       - Beautiful hexagonal battlefield
✅ MOUSE CONTROLS        - Point-and-click interface
✅ COLOR-CODED TEAMS     - Blue vs Red team colors
✅ HEALTH BARS           - Visual HP indicators
✅ CHARACTER CLASSES     - 6 classes with unique colors
✅ PHASE DISPLAYS        - Selection, Placement, Battle phases
✅ REAL-TIME NETWORKING  - Same server protocol as text client

BUILT FILES:
================================================================================

./bin/HexGameServer      - Game server (works with both clients)
./bin/HexGameClient      - Text-based client (original)
./bin/HexGameSFML        - NEW! SFML graphical client

HOW TO RUN:
================================================================================

1. Start the server:
   $ ./bin/HexGameServer

2. Option A - Text clients (original):
   Terminal 2: $ ./bin/HexGameClient
   Terminal 3: $ ./bin/HexGameClient

2. Option B - SFML graphical clients:
   Terminal 2: $ ./bin/HexGameSFML
   Terminal 3: $ ./bin/HexGameSFML

2. Option C - Mixed (1 text + 1 SFML):
   Terminal 2: $ ./bin/HexGameClient
   Terminal 3: $ ./bin/HexGameSFML

All combinations work! The server doesn't care which client type connects.

CONTROLS:
================================================================================

SFML Client:
  - MOUSE CLICK: Select characters, place on grid, issue commands
  - SPACEBAR: Lock in your turn (battle phase)
  - ESCAPE: Quit game

Text Client:
  - TYPE COMMANDS: select, place, move, skill, ready, quit
  - See in-game help with 'help' command

VISUAL GUIDE:
================================================================================

Team Colors:
  🔵 Blue outline = Player 0 (left side, q < 0)
  🔴 Red outline  = Player 1 (right side, q > 0)

Character Classes:
  🟥 Red    = Warrior
  🟪 Purple = Mage  
  🟨 Yellow = Healer
  ⬜ Gray   = Rogue
  🟫 Brown  = Tank
  🟩 Green  = Summoner

Grid Zones:
  Dark blue  = Player 0 territory (left)
  Dark red   = Player 1 territory (right)
  Gray       = Neutral middle

ARCHITECTURE:
================================================================================

The SFML client maintains the same game logic as the text client:
  - Same network protocol (Boost.Asio)
  - Same game state synchronization
  - Same server-side validation
  - Different presentation layer only

Both clients can play against each other seamlessly!

GAMEPLAY FLOW:
================================================================================

1. SELECTION PHASE:
   SFML: Click on 6 character boxes to select 3 classes
   Text: Type "select 0" through "select 5"

2. PLACEMENT PHASE:
   SFML: Click hexes on your side to place characters
   Text: Type "place <class_id> <q> <r>"

3. BATTLE PHASE:
   SFML: Click to command, Space to lock turn
   Text: Type commands, "ready" to lock turn

4. GAME OVER:
   Winner announced, game ends

TECHNICAL DETAILS:
================================================================================

Rendering:
  - 60 FPS game loop
  - Flat-top hexagon layout
  - SFML 3.0.2 modern C++ API
  - Async rendering + network I/O

Networking:
